//PÁGINA UTILIZADA PARA ESTUDO

VARIAVEIS:

  • var - não recomendado para utilização
  • let - variável com escopo de bloco
  • const - variável apenas para leitura, não podendo atualizar o valor depois
PARA COLOCAR CARACTERES DA STRING ( \ ):
  • \n - nova linha
  • \\ - exibe: a \
  • \' exibe: '
  • \" - exibe: "

OPERADOR DESCRIÇÃO
== valor igual
=== valor e tipo iguais
!= valor diferente de
!== valor e tipo diferentes de
<, <= menor (menor igual) a
>, >= maior (maior igual) a
&& "E" lógico (retornar true se os testes forem verdadeiro)
|| "OU" lógico (retorna true se 1 dos testes foram verdadeiro)
! "NÃO" (inverte o booleano)
